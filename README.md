# Carousel-showcasing-products

## Description

It's project to show how I work with React and Tailwindcss to create simple aplication;

## Technology used

1.  Project was setup with [Vite](https://vite.dev/)
2.  Main library [React](https://react.dev/)
3.  For styling and animations [Tailwindcss](https://tailwindcss.com/)
4.  Testing with [Vitest](https://vitest.dev/)
5.  For storage used [React-Context](https://react.dev/reference/react/createContext) (enough for this level of application)
6.  For hosting used [GitHub Pages](https://docs.github.com/en/pages)
7.  AI helper [Gemini CLI](https://github.com/google-gemini/gemini-cli)

## Usage

Start the application with the following command:

```bash
npm run dev
```
